Dieses Package beschreibt die fundamentalen Attribute.

Weitere Informationen im Wiki.

Die Klasse Attribut ist die fundamentale Klasse von der alle Attribute ableiten. Von den Klassen auf dieser Ebene können keine Objekte gebildet werden. Alle Klassen hier sind abstract deklariert. Mit einem Attribut wird aus einem BO den Bezug zu einem Daten-Attribut in einem POJO Objekt hergestellt. In einem BO werden verschiedene von Attribut abgeleitete AttributKlassen zu einer Komposition zusammen gestellt.
